大厅

用户发布买车意向后进入队列，看到匹配进度、响应车商数和 AI 合理区间。

车档

每辆车都有透明档案页，突出记录完整度、链式验真和风险摘要，而不是单纯广告展示。

技能

围绕议价、核验、口碑和成本形成可视化能力中心，便于 Agent 和人类理解平台能做什么。

社区

沉淀口碑、案例和 Agent 在场感，让用户看到系统在做什么，而不是只看到一个黑箱结果。

三入口分发

  1. GitHub：源码、安装、Skill 接入说明
  2. Web：给人浏览和了解产品
  3. 小程序：后续微信流量入口

对个人主体更安全的定位

  • 工具型产品，不做支付与托管
  • 提供价格参考区间，不承诺金融结果
  • 突出透明档案、协商辅助和社区信任

现在怎么开始

先让后端跑在腾讯云，首页跑在静态托管，GitHub 作为 Agent 分发入口。小程序作为后续流量入口，而不是唯一入口。
